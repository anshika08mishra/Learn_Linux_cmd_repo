## uname
  - Displays system information about the Linux machine (kernel, OS, architecture).
  - Syntax: ```uname [options]```
  - Example: ```uname -a```
  - Sample Output (simplified): ```Linux ubuntu 5.15.0-84-generic x86_64 GNU/Linux```
  - Useful Options:
       - ```-a``` → Show all system information ✅
       - ```-r``` → Show kernel release
       - ```-m``` → Show machine hardware (architecture)
       - ```-s``` → Show kernel name

## uptime
   - Shows how long the system has been running along with current load.
   - Syntax: ```uptime```
   - Example: ```uptime```
   - Sample Output (simplified): ```10:30:25 up 2 days, 3:15,  2 users,  load average: 0.25, 0.30, 0.28```
   - Output Meaning:
       - ```10:30:25``` → Current system time
       - ```up 2 days, 3:15``` → System running time
       - ```2 users``` → Logged-in users
       - ```load average``` → System load (last 1, 5, 15 minutes)
